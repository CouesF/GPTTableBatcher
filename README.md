# GPTTableBatcher

## 介绍
这是一款为社畜开发的轻量级工具（单网页），用于通过表格批量调用GPT模型，适用于批量文本生成和调用。它允许用户上传表格数据，通过GPT模型批量处理这些数据，并查看结果。这个工具的设计注重便携性和易用性，适合于需要快速部署和使用的场景。

## 特性
- 批量调用GPT模型处理表格数据
- 简单易用的Web界面
- 支持在线公式编辑
- 支持上传和下载Excel文件
- 轻量级，无需复杂配置，易于部署

## 安装和运行
- 直接下载网页文件使用即可
